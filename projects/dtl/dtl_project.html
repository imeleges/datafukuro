Short Description

Dubai Tenancy Lookup is a Python and Streamlit app that utilizes open data from the Dubai Land Department to provide renters in the Marsa Dubai area with property information, rental price history, and comparisons to similar properties in the same building.

Description

Dubai Tenancy Lookup is an online application that allows users to access and analyze data on rental properties in the Marsa Dubai area. Developed using Python and Streamlit, this app provides users with valuable insights into their rental contracts, such as property information, rental price history, and comparisons to other similar properties in the same building. The app is built on open data from the Dubai Land Department, which provides information on the rental market in this region.

One of the features of this app is its user-friendly interface, which allows users to easily enter their rental contract number and view all relevant information in one place. In addition to property information and rental history, the app also provides information on the building and its location, showing a pin on a map for easy reference. This information is useful for renters who want to understand the rental market in the Marsa Dubai area and make informed decisions about their rental contracts.

Overall, Dubai Tenancy Lookup is an excellent tool for anyone looking to rent a property in the Marsa Dubai area. By providing easy access to important rental information and data analysis, this app empowers renters to make informed decisions and negotiate fair rental prices. With its user-friendly interface and reliable data sources, this app is a valuable resource for anyone navigating the rental market in Dubai.

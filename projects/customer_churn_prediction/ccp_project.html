Brief description

By analysing customer behaviour data, the project applies ML techniques to identify customers who are likely to leave the bank. Creating a model on predicting customer churn using comprehensive data analysis and machine learning techniques helping to come up with plans to keep them.

Description

The project begins with an exploratory data analysis (EDA) of the customer data, providing insights into various features and their potential relationships with customer churn. Next, the project proceeds with data preprocessing, involving data cleaning, handling missing values, and categorizing variables.

Various machine learning models, including logistic regression, random forest, and gradient boosting, are trained and evaluated. The models' performance is compared using metrics like accuracy, precision, recall, and F1-score. The project concludes by selecting the best model based on the evaluation metrics and discussing potential strategies for customer retention.

The project successfully predicts customer churn using machine learning models, providing with valuable insights to potentially improve customer retention. The exploratory data analysis, comprehensive preprocessing, and careful model selection all contribute to the project's success. The results obtained could help in strategizing customer retention policies.

Brief description

This project is focused on the goal to build a model for a used car trading service to attract new customers. This model will help users quickly understand the market value of their cars using historical data.

Description

The project starts with data preparation and cleaning. The data includes various technical specifications, configurations, and prices of used cars. The model features include the car's type, year of registration, gearbox type, power, model, mileage, fuel type, brand, repair history, and more. The target feature is the price of the car.

The data was cleaned to fill in missing values and remove duplicates. Outliers were identified and treated to ensure the quality of the data. The categorical features were then encoded, and the features were scaled before model training.

Several models, including Linear Regression, Random Forest, XGBoost, and LightGBM, were trained using the prepared data. Each model's performance was evaluated based on the Root Mean Square Error (RMSE) and the time taken for training and prediction.

The models' evaluation revealed differences in both their prediction quality and speed. The RandomForestRegressor model was the fastest, taking approximately one minute to train and predict, but it did not provide the best RMSE score. The XGBoost and LightGBM models provided better RMSE scores but took longer to train. Notably, the LightGBM model provided the best RMSE score when trained on a version of the data with categorical columns but took the longest to train.

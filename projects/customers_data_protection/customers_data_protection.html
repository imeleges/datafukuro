Brief description

The aim of this project was to develop a method to protect the personal information of clients, making it difficult to recover their personal data.

Description

The project unfolded in a series of clearly defined stages.

Initially, the necessary data was loaded and the essential libraries for the project were imported. The data was structured appropriately, setting a solid foundation for the subsequent phases of the project.

Following this, an in-depth exploration into the principles of matrix multiplication took place, which served as a cornerstone for the project. Leveraging these principles, a transformation algorithm was designed.

The final phase encompassed testing and verifying the effectiveness of this algorithm. Thanks to this structured approach, the project transitioned smoothly from data loading, through algorithm development, to its final verification.

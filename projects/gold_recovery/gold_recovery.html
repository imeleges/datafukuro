Brief description

This project aims to develop a machine learning model to predict the gold recovery rate from gold ore. The model will help optimize production processes and avoid running operations with unprofitable characteristics.

Description

The project is divided into three main sections:

  • Data Preparation
  • Exploratory Data Analysis
  • Model Building and Training

The Data Preparation stage involves cleaning and preparing the data for subsequent analysis. The Exploratory Data Analysis stage includes investigating the concentration of metals at various purification stages, comparing the distribution of raw material grain sizes across training and test datasets, and investigating the total concentration of substances at different stages.

In the Model Building and Training stage, several machine learning models, including Linear Regression, Decision Tree, and Random Forest, are evaluated. The Symmetric Mean Absolute Percentage Error (sMAPE) is used as the evaluation metric. Hyperparameters for these models are tuned using grid search and cross-validation.

The RandomForestRegressor model delivers the best performance on the training set, with an sMAPE score of 10.334961%. On the test set, this model achieves an sMAPE score of 11.583625%, which is comparable to the performance of the simple model. However, the LinearRegression model achieves a slightly better result with 11.062478% on the test set.

Despite this, the recommendation is retaining the RandomForestRegressor model as it provides a good balance between complexity and performance.

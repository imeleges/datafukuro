Brief description

This project involves developing a model to analyze customer behavior and recommend new tariffs for the cellular service provider. The model employs machine learning techniques to predict the most suitable tariff for each customer, based on their usage data.

Description

The project begins by loading and studying the dataset, which contains information about clients who have already switched to the new tariffs. The primary goal is to build a classification model with the highest possible accuracy, which will determine the most suitable tariff for each client, either Smart or Ultra.

The data is split into different subsets for model training, validation, and testing. Several machine learning models, including Decision Tree, Random Forest, and Logistic Regression, are investigated. Each model's performance is evaluated based on the accuracy metric, with a focus on hyperparameter tuning to achieve the best performance.

For instance, the optimal tree depth for Decision Tree and Random Forest models is determined through a range of values. Similarly, the number of estimators is also tuned for the Random Forest model.

The project successfully develops a recommendation system that utilizes user behavior data to suggest the most appropriate mobile tariff. This system aids the cellular service provider in providing tailored services, improving customer experience, and potentially increasing revenue.
